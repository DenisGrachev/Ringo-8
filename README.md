# Ringo Engine
64x48 Gfx Lib For ZX Spectrum 128KB base–≤ on my render engine from [Ringo](https://zxonline.net/game/ringo)

# Quick Start

Download sources, put sjasmplus.exe in the right folder and run !compile.bat
